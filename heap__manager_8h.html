Detailed Description

Heap manager allocator interface; this interface routes the allocator calls based on the HEAP ID.

Copyright
Please read the Exosite Copyright: copyright

Features:

  • Support for several heaps
  • Thread safe
  • In debug mode instrumentation is added
  • Support for memory IDs
