int32_t system_thread_create ( system_thread_t *  thread,
const system_thread_attr_t *  attr,
void *(*)(void *)  start_routine,
void *  param 
)

Creates a new thread and makes it executable. This routine can be called any number of times from anywhere within your code.

Parameters
[out]thread- An opaque, unique identifier for the new thread returned by the subroutine.
[in]attr- An opaque attribute object that may be used to set thread attributes. You can specify a thread attributes object, or NULL for the default values.
[in]start_routine- The C routine that the thread will execute once it is created.
[in]arg- A single argument that may be passed to start_routine. It must be passed by reference as a pointer cast of type void. NULL may be used if no argument is to be passed.
Returns
Error code. (No error = 0)
