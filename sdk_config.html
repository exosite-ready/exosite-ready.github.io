Configuration

config_port.h header contains settings that are able to tune the Exosite Ready SDK to be suitable to the application and the environment.

There are a couple of groups of these settings:

  • config_log - Adjust log verbosity / turn on/off log level in compile time
  • config_transmission - Adjust transmission queue size and response timeout
  • config_sdk_modules - Include/Exclude SDK modules in/from compilation
  • configurator_config - Configurator server settings
  • heap_config - Heap configuration
  • config_wifi - WiFi Network settings (SSID, Password)
  • config_provisioning - Provisioning (device activation) settings
  • config_other - Common settings
  • config_environment - Environment dependent settings
Warning
In most of the cases you do not need to modify these settings.
