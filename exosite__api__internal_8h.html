typedef void(* on_subscribe_clb) (int32_t, enum application_protocol_type, struct exo_request_args *, const char *)

Subscribe callback.

Subscribe callbacks are handled in 2 steps:

  • 1st the exosite_api will handle the callback and determine whether a new subscribe request needs to be initiated or not;
  • 2nd the exosite_api subscribe callback handler will call the callback provided by the user if necessary.
