[in]sizeThe size of the buffer
[out]result_codeThe extracted response code (Valid if a HTTP header was found)
[out]content_lengthThe extracted content length (Valid if a HTTP header was found)
[out]payload_indexThe position of the payload (Valid if a HTTP header was found)
Returns
